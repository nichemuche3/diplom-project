# API - Автоматизация закупок

Backend-приложение для автоматизации закупок в розничной сети.

## О проекте

REST API сервис для автоматизации закупок с функционалом:
- Регистрация и авторизация пользователей (JWT)
- Управление товарами и категориями
- Корзина и оформление заказов
- Импорт товаров из YAML
- Отправка email уведомлений
- Документация Swagger/ReDoc http://127.0.0.1:8000/swagger/ или http://127.0.0.1:8000/redoc/
